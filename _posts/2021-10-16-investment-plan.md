## 계획

### 예측

|                | S+ (기울기>0)   | S0 (기울기=0)   | S- (기울기<0)         |
| :------------- | :------------ | :------------- | :------------------ |
| V+ (진동 강함)   | S+/V+         | S0/V+          | 인버스 거래 (S+/V+)    |
| V0 (진동 약함)   | S+/V0         | 거래 대상 아님    | 인버스 거래 (S+/V0)    |

### 대응

| 분류      | 판단          | 행동                      |
| :-       | :-           | :-                       |
| S+/V+    | 위로 진동      | 부분 매도                  |
|          | 아래로 진동     | 추가 매수                 |
| S0/V+    |              | 정방향/인버스 동시 보유       |
|          | 위로 진동      | 정방향 매도 <br> 인버스 매수    |
|          | 아래로 진동     | 정방향 매수 <br> 인버스 매도    |
| S+/V0    | 기울기<0       | 매도                     |
